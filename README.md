PC Room Linux Maker
===

ldif files for openldap-based computer room linux system

## Requirement
Open LDAP Server
NFS Server

Open LDAP Client
NFS Client

autofs Daemon

## Usage
See `man ldapadd` , `man ldapdelete` , `man ldapmodify` and so on.
